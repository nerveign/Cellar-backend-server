## Setup .env file

```dotenv
PORT=...
DATABASE_URL=...
JWT_SECRET=...
JWT_EXPIRES_IN=...
NODE_ENV=development

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```

## Run the app (development)

```bash
npm run dev
```

## Build the app

```bash
npm run build
```

## Run the app

```bash
npm run start
```
